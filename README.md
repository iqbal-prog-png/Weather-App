# ☁️ Weather App

A simple and beautiful weather application built with HTML, CSS, and JavaScript that fetches real-time weather data from the OpenWeatherMap API.

## 📸 Features

- 🌍 Search weather for any city worldwide
- 🌡️ Display current temperature in Celsius
- 💨 Show wind speed, humidity, pressure, and visibility
- 🎨 Beautiful gradient UI with smooth animations
- 📱 Responsive design for all devices
- ⚡ Real-time weather data
- 🎭 Dynamic weather icons based on conditions

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling and animations
- **JavaScript (ES6+)** - Functionality and API integration
- **OpenWeatherMap API** - Weather data

## 📋 Prerequisites

Before running this project, you need:

- A web browser (Chrome, Firefox, Safari, etc.)
- An API key from [OpenWeatherMap](https://openweathermap.org/api)
- Internet connection
